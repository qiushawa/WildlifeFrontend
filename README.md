# Wildensity Frontend
本專案是一個基於 React 和 Vite 開發的野生動物監測前端系統。
## 專案結構
```
wildensity-frontend/
├── public/                # 靜態資源
├── src/                   # 原始程式碼
│   ├── components/        # React 組件
│   ├── pages/             # 頁面組件
│   ├── assets/            # 圖片、樣式等資源
│   └── App.tsx            # 應用程式入口
├── eslint.config.js       # ESLint 配置
├── tsconfig.app.json      # TypeScript 應用程式配置
├── tsconfig.node.json     # TypeScript Node 配置
├── vite.config.ts         # Vite 配置
└── README.md              # 本文件
```
